# HTML e CSS - Curso em Vídeo

Este repositório contém os desafios e projetos desenvolvidos durante o curso de **HTML e CSS** do [Curso em Vídeo](https://www.cursoemvideo.com/).

## Sobre o Curso

O curso de HTML e CSS do **Curso em Vídeo**, ministrado pelo professor **Gustavo Guanabara**, é uma excelente introdução ao desenvolvimento web. Ele aborda desde os conceitos básicos de HTML e CSS até tópicos mais avançados, como responsividade e boas práticas de desenvolvimento.
